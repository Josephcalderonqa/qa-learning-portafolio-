# Ejercicios de LEFT JOIN

Esta carpeta contiene ejercicios prácticos de SQL utilizando **LEFT JOIN**, enfocados en escenarios reales de QA Backend para sistemas bancarios y fintech.

## Objetivo

Aprender a identificar registros sin relación en otras tablas, una habilidad muy utilizada para detectar inconsistencias, validar integraciones y encontrar información faltante durante las pruebas de backend.

## Ejercicios

1. Cuentas bancarias sin transferencias.
2. Wallets sin recargas móviles.
3. Tarjetas sin retiros en cajeros ATM.
4. Transacciones QR sin validación antifraude.
5. Pagos PSE sin confirmación bancaria.
6. Transferencias reversadas sin ticket de soporte.
7. Desembolsos aprobados sin movimiento contable.

## Habilidades practicadas

- LEFT JOIN
- IS NULL
- Relaciones entre múltiples tablas
- Validaciones de integridad de datos
- Casos de uso reales de QA Backend
